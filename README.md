## Binary Tree: Find Max

You are given a binary tree. Implement the method *findMax* which returns the maximum value. 

*E.g:*
```
        13
       /   \
      3     5
     /    /  \
    2    16   17
             /
            13
            
result : 17
```

The class `TreeNode` is available for you:

``` java
    class TreeNode {
        private int value;
        private TreeNode left;
        private TreeNode right;
        // ...
    }
```
