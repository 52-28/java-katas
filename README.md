# Roman to Arabic

Create a method which converts roman numeral to arabic.

If the argument is *null*, empty or contains non roman numeral, throw an `IllegalArgumentException`.

E.g :

```
I       -> 1
III     -> 3
IV      -> 4
V       -> 5
VIII    -> 8
IX      -> 9
X       -> 10
IXC     -> 89
CXI     -> 111
DCLXVI  -> 666
M       -> 1000
```
