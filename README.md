# String Compare

Create a method which returns the number of identical characters at same position, from two `String` of same length.

If the two arguments doesn't have the same length, return -1 instead.

*E.g:*

```
origin   compare     result
"a"      "a"         1
"a"      "b"         0
"aa"     "ba"        1
"cassis" "castor"    3
"tacos"  "poulpe"   -1
```
