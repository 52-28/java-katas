# Decimal to hexadecimal

Create a method that, given a positive decimal, returns a string representation of an hexadecimal value.

If the argument is negative, throw an `IllegalArgumentException`.

*E.g:*
```
    0 : 0
    1 : 1
    9 : 9
   10 : A
   11 : B
   15 : F
   16 : 10
   17 : 11
  255 : FF
20268 : 4F2C
```
